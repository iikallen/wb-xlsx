# Wildberries Parser

Минимальный CLI-парсер для `wildberries.ru`, который:

- собирает полный каталог товаров по поисковому запросу;
- сохраняет результат в `XLSX`;
- делает второй `XLSX` с фильтром `рейтинг >= 4.5`, `цена <= 10000`, `страна производства = Россия`.

## Что выгружается

- ссылка на товар
- артикул
- название
- цена
- описание
- ссылки на изображения через запятую
- все характеристики с сохранением структуры в JSON
- название селлера
- ссылка на селлера
- размеры товара через запятую
- остатки по товару
- рейтинг
- количество отзывов

## Запуск

```bash
npm install
npm start
```

По умолчанию используется запрос:

```text
пальто из натуральной шерсти
```

Готовые файлы сохраняются в папку `output/`:

- `output/wildberries_catalog_full.xlsx`
- `output/wildberries_catalog_filtered.xlsx`

## Полезные параметры

```bash
npm start -- --query "пальто из натуральной шерсти"
npm start -- --dest -1257786
npm start -- --limit 100
npm start -- --headless false
npm start -- --output-dir ./output
```

## Примечания

- По умолчанию скрипт запускает обычный браузер Edge/Chrome, потому что у Wildberries есть антибот-проверка.
- Для характеристик используется структурированный JSON из `card.json`, чтобы не терять группировку полей.
- `dest=-1257786` соответствует российской выдаче.
